create(bucket, options) → {Driver}

Source:
  • index.js, line 299

Create a Driver object by wrapping the Couchbase bucket and creates a new Driver instance and
adds Promise support to the instance.

Example
const couchbase = require('couchbase');
const create = require('couchbase-driver').create;
const cluster = new couchbase.Cluster('couchbase://127.0.0.1');
const bucket = cluster.openBucket('default');
const driver = create(bucket);
